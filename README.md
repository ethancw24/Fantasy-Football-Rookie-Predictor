Fantasy Football Rookie Predictor
A machine learning pipeline that predicts which incoming NFL rookies will become fantasy football stars — using college stats, NFL Combine measurements, and NFL team context.
Targets half-PPR and full-PPR scoring only.

Getting Started
1. Install UV
UV is the package manager this project uses. Install it once globally:
powershellpip install uv
2. Clone the repository
bashgit clone https://github.com/your-username/fantasy-football-rookie-predictor.git
cd fantasy-football-rookie-predictor
3. Install dependencies
UV creates the virtual environment and installs everything in one command:
bashuv sync
4. Run the pipeline
bash# Normal run — uses cached data if available
uv run python main.py

# Force a fresh fetch from APIs (use at the start of a new NFL season)
uv run python main.py --refresh

# Run Stage 1 only, skip cleaning
uv run python main.py --skip-cleaning

Notes for Development

data/raw/ and data/filtered/ are auto-generated — do not commit them to git.
uv.lock should be committed to git — it ensures every machine uses the exact same package versions.
NaN values from missing combine or college data are preserved intentionally through Stages 1 and 2; the ML model's gridsearch handles them in Stage 3.
College-to-NFL season alignment uses draft_year - 1 for drafted players and first_nfl_season - 1 for undrafted free agents.
